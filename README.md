## Selected Work

### Flexible TPM / Atlassian Forge Jira Table App

Jira 이슈를 단순 목록으로 조회하는 데 그치지 않고, 사용자가 필터링, 정렬, 핀, 그룹핑, 계산 컬럼을 통해 업무 데이터를 직접 가공할 수 있는 테이블 앱을 개발하고 있습니다.

- Atlassian Forge Custom UI, React, TypeScript, Vite, Atlaskit, AG Grid
- 포뮬러 컬럼 생성/수정 모달, 함수/필드 검색, 괄호 자동 닫기, 함수 삽입 흐름 구현
- 계산 컬럼이 정렬/필터/핀/그룹핑 흐름에서 일반 컬럼처럼 동작하도록 상태 관리 로직 개선
- 셀 스켈레톤, 로딩 진행바, 필드 프리페치를 통해 데이터 로딩 중 체감 안정성 개선

### Crypto Exchange Admin

가상화폐 거래소 운영에 필요한 회원, 입출금, AML, 리포트, 자산, 권한 관리 기능을 하나의 백오피스 흐름으로 통합한 관리자 시스템을 개발했습니다.

- Next.js, TypeScript, Recoil, React Query, Mantine UI
- 관리자 시스템 프론트엔드 약 70% 담당
- 회원, 입출금, AML, 자산, 리포트, 권한 관리 화면 구현
- 역할 기반 접근 제어와 운영자 조치 흐름을 고려한 화면 구성

### AI Labeling Platform

AI 학습용 라벨링 데이터를 관리하는 관리자 사이트와 실제 작업자가 사용하는 라벨링 툴을 개발했습니다.

- React, TypeScript, Redux, React Query, Material UI, Tailwind CSS
- 작업 배정, 프로젝트 관리, 작업자 권한, 정산, 통계 화면 구현
- Canvas 기반 라벨링, 지우개, 되돌리기, 히스토리, Issue, Log 기능 구현
- 차트 라이브러리 기반 대시보드와 작업 현황 시각화
